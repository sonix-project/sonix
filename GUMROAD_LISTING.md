## Full Description

### Tired of building social features from scratch?

**Sonix** is a complete, production-ready social media platform. You get a Laravel 13 API backend + React Native (Expo) mobile app with every feature your users expect.

### What's Included

**Backend (Laravel 13)**
- 63+ REST API endpoints
- 17 database models, 34 migrations
- Sanctum token authentication
- Real-time WebSocket messaging
- Image/video upload with compression
- Cloudinary cloud storage integration
- PostgreSQL + Redis support
- Docker + Railway one-click deploy

**Mobile App (React Native / Expo)**
- 24 beautiful screens
- 3D animated tab bar with glassmorphism
- Video backgrounds on auth screens
- Instagram-style story creation
- Real-time chat with typing indicators
- Push notifications
- Arabic + English (300+ translations)
- Dark mode "Midnight Luxe" theme

### Features
- User registration & login (Sanctum tokens)
- User profiles with avatars, bio, followers
- Private/public account toggle
- Create posts (text, photo, video)
- Like, comment, bookmark posts
- Share posts to conversations
- Instagram-style stories (photo, video, text, drawing, stickers)
- Story highlights
- Story viewer with sound toggle
- Real-time messaging (text, image, voice)
- Typing indicators & online status
- Follow/unfollow with follow requests
- User search
- Block/report users
- Push notifications
- i18n (Arabic + English)
- Image compression
- Cursor-based pagination

### Tech Stack
- **Backend:** Laravel 13, PHP 8.4, PostgreSQL, Redis, Sanctum
- **Mobile:** React Native, Expo SDK 56, AsyncStorage
- **Deploy:** Railway (backend), EAS Build (APK)
- **Storage:** Cloudinary (optional) + local fallback
