## Product Overview

### What is Sonix?

Sonix is a **full-featured social media platform** that combines the best features of:

| Platform | Sonix Feature |
|----------|---------------|
| Instagram | Posts, Stories, Highlights, Explore, Reels |
| Snapchat | Vanish Mode, Story Reactions, Profile Visitors |
| Twitter/X | Hashtags, Mentions, Trending |
| WhatsApp | Voice Messages, Group Chat, Typing Indicators |
| TikTok | Short Video Reels, Video Posts |

### Unique Selling Points

1. **Full RTL Support** — Complete Arabic + English with automatic text direction
2. **Voice Messages** — Record, send, and play voice messages with animated waveform
3. **Vanish Mode** — Messages auto-delete after being seen (like Snapchat)
4. **Group Chat** — Create groups, add/remove members, group messaging
5. **Story System** — Create stories with stickers, drawing, music, analytics
6. **Profile Visitors** — See who viewed your profile
7. **Profile Templates** — Customizable profile themes
8. **2FA Security** — Two-factor authentication via email

## Deployment Options

### Option 1: Docker (Recommended)
```bash
docker build -t sonix .
docker run -p 80:80 sonix
```

### Option 2: Traditional Server
```bash
# Backend
composer install --no-dev
php artisan config:cache
php artisan migrate --force

# Frontend
cd expo-app
npm install
npx expo start
```

### Option 3: Cloud Platforms
- **Backend:** Railway, DigitalOcean, AWS, Heroku
- **Mobile:** EAS Build (Expo Application Services)
